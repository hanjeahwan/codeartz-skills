# Agent Evolve 展开回执

仅在 `validation.md` 判定为未解决冲突或处理失败后读取并使用：

```text
Agent Evolve｜<待审核 | 处理失败>
候选来源：<来源 | 不适用>
直接原因：<冲突或失败>
证据：<决定结论的项目事实>
目标位置：<项目相对路径 | 不适用>
建议规则：<精确提案 | 不适用>
下一步：<仅待审核；解除阻塞的裁决或动作>
```

- `处理失败` 不输出下一步；直接原因和证据说明已知恢复条件，未知时如实标记。
- 读取失败时不得声称完成查重或查冲突。
- 写入或重读失败时不得声称已沉淀。
- Agent Evolve 失败不得阻止当前任务继续完成。
